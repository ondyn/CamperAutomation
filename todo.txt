1) tilt meter
homeassistant plugin which will use accelerometer via termux:api and will have this functionality:
    - this integration will be used to detect car tilt (pitch, roll) based on phones accelerometer/gyroscope data. It will be used for level up the car. User can set axle track and wheelbase. User can set zero (will perform, when car will be level).
    - it will show pitch and roll in degrees and calculate hight for each wheel, how much to lift the specific wheel. only wheel height adjustment is possible, so the lowest wheel will be used as reference. It will also show if the car is tilted to the left or right (roll) and front or back (pitch). 
    - sesor data consumption and calculation will be done only on user request (or if the screen in homeassistat companion app is visible) to save battery. User can set the update interval for the sensor data.
